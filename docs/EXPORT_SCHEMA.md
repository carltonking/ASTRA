# ASTRA Export Schema

## Format

A single self-contained Python file (`.py`). Not an archive — one file contains everything.

```python
"""<metadata docstring>"""

# GRADUATION CERTIFICATE
# Certificate ID: <uuid>
# ...

# Risk Limits
STOP_LOSS = <float>
TAKE_PROFIT = <float>
MAX_POSITION_SIZE = <float>
MAX_POSITIONS = <int>

# Parameters
<PARAM_NAME> = <value>

STRATEGY_METADATA = { <dict> }

<strategy code, BaseStrategy inlined>

if __name__ == "__main__":
    <demo harness with random data>
```

## Sections (in order)

### 1. Docstring
Multiline Python docstring containing:
- Strategy type, symbols, timeframe
- Risk parameters (stop-loss, take-profit, max position size, max positions)
- Limitations (5 standard limitations)
- Export timestamp, checksum
- Disclaimer

### 2. Certificate Header
Comment block with:
- Certificate ID, Session ID, Spec ID
- Issued date
- Strategy type
- Optimization cycles
- Per-gate results: name, status (PASSED/FAILED), actual value, threshold, gap

### 3. Risk Limits
Module-level constants in ALL_CAPS:
- `STOP_LOSS` (float)
- `TAKE_PROFIT` (float)
- `MAX_POSITION_SIZE` (float)
- `MAX_POSITIONS` (int)

### 4. Strategy Parameters
Module-level ALL_CAPS constants for each strategy parameter:
- One constant per parameter from `initial_parameters`
- Uppercase version of parameter name (e.g. `fast_window` → `FAST_WINDOW`)

### 5. STRATEGY_METADATA Dict
Dictionary with:
- `certificate_id`, `spec_id`, `session_id`
- `strategy_type`, `strategy_class_name`
- `symbols`, `timeframe`, `hypothesis`
- `initial_parameters`, `parameter_bounds`
- `backtest_sharpe`, `backtest_return`, `backtest_max_drawdown`
- `backtest_annualized_return`, `backtest_win_rate`
- `paper_return`, `paper_sharpe`, `paper_max_drawdown`, `paper_win_rate`, `paper_days_deployed`
- `degradation_category`
- `version` (always `"1.0"`)
- `disclaimer`

### 6. Strategy Code
Sanitized strategy class with `BaseStrategy` ABC inlined:
- `from astra.*` imports stripped
- `ABC` and `@abstractmethod` decorators stripped from BaseStrategy
- Original strategy class preserved with its `generate_signals`, `get_parameters`, `get_parameter_bounds` methods

### 7. Demo Harness
`if __name__ == "__main__":` block that:
- Generates random price data
- Instantiates strategy with exported parameters
- Calls `generate_signals()`
- Prints summary statistics

## Validation

Export files are validated by `ExportValidator` which checks for:
- Presence of `GRADUATION CERTIFICATE` header
- Presence of `STRATEGY_METADATA` dictionary
- Presence of disclaimer text (contains "past performance")
- Presence of Limitations section (contains "Limitations")

## Checksum

SHA-256 hash of the entire file content. Stored in the `ExportPackage` dataclass.

## Example

```python
"""ASTRA Exported Strategy: Trend Following
Symbols: SPY, QQQ
Timeframe: daily

Risk Parameters:
  Stop Loss: 5.0%
  Take Profit: 10.0%
  Max Position Size: 10.0%
  Max Positions: 5

Limitations:
  1. Past performance does not guarantee future results
  2. This strategy was validated only in specific market conditions
  3. Paper trading does not account for slippage, fees, or liquidity constraints
  4. Live trading may produce substantially different results
  5. This certificate does not constitute financial advice

Disclaimer:
ASTRA research results are not profitability guarantees.
Past performance does not predict future results.
This file was generated by ASTRA for research purposes only.

Export Timestamp: 2026-05-21T12:00:00+00:00
Checksum: a1b2c3d4e5f6...
"""

# GRADUATION CERTIFICATE
# Certificate ID: abc-123
# ...

STOP_LOSS = 0.05
TAKE_PROFIT = 0.10
MAX_POSITION_SIZE = 0.10
MAX_POSITIONS = 5

FAST_WINDOW = 20
SLOW_WINDOW = 50

STRATEGY_METADATA = {
    "certificate_id": "abc-123",
    "version": "1.0",
    ...
}

# ... strategy code ...

if __name__ == "__main__":
    # Demo harness
    ...
```

## Versioning

Current format version: `1.0` (stored in `STRATEGY_METADATA.version`). Future format changes will increment this version number.
